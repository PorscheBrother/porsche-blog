第一部分：JavaScript 基礎入門

為什麼學 JavaScript？

JavaScript 是網頁的靈魂，原因：

  • 無處不在： 所有瀏覽器都支援
  • 全端開發： 前端 + 後端（Node.js）都能寫
  • 就業市場大： 網頁開發必備技能
  • 生態豐富： 無數框架和套件（React, Vue, Angular）

🌐 JavaScript ≠ Java

就像「Car」和「Carpet」一樣，名字相似但完全不同！JavaScript 是腳本語言，Java 是編譯語言。

運行 JavaScript 的三種方式

方式 1：瀏覽器控制台（最簡單） 1. 打開任意網頁 2. 按 F12 打開開發者工具 3. 點選 Console（控制台） 4. 輸入：console.log("Hello, World!") 5. 按 Enter 方式 2：HTML 內嵌 <script> alert("Hello!"); </script> 方式 3：外部檔案 <script src="app.js"></script>

你的第一個程式

// 方法 1：輸出到控制台 console.log("Hello, World!"); // 方法 2：彈出警告框 alert("我是 Porsche哥"); // 方法 3：寫入網頁 document.write("<h1>Hello</h1>");

💡 提示：使用 Chrome 瀏覽器 + VS Code 編輯器，有強大的開發者工具和 IntelliSense 自動完成。

第二部分：核心概念

1. 變數宣告（Variables）

JavaScript 有三種宣告變數的方式：

// var - 舊方式（不推薦） var name = "Porsche哥"; // let - 現代方式（可重新賦值） let age = 5; age = 6; // OK // const - 常量（不可重新賦值） const PI = 3.14159; PI = 3; // Error! // 推薦：優先使用 const，需要改變時用 let

⚠️ 避免使用 var：var 有函數作用域，容易造成 bug。let 和 const 有塊級作用域，更安全。

2. 資料類型

  • 字串 (String)： 文字資料
  • 數字 (Number)： 整數或浮點數
  • 布林 (Boolean)： true 或 false
  • 物件 (Object)： 鍵值對集合
  • 陣列 (Array)： 有序列表
  • null： 空值
  • undefined： 未定義
// 字串 let text = "Hello"; // 數字 let number = 42; let decimal = 3.14; // 布林 let isTrue = true; let isFalse = false; // 物件 let person = { name: "Porsche哥", age: 5, city: "Taipei" }; // 陣列 let fruits = ["apple", "banana", "orange"]; // null 和 undefined let empty = null; let notDefined = undefined;

3. 運算子

// 算術運算子 let a = 10; let b = 3; a + b // 13 a - b // 7 a * b // 30 a / b // 3.333... a % b // 1（餘數） a ** b // 1000（次方） // 比較運算子 a == b // false（相等） a === b // false（嚴格相等） a != b // true（不相等） a !== b // true（嚴格不相等） a > b // true a < b // false // 邏輯運算子 true && false // false（AND） true || false // true（OR） !true // false（NOT）

4. 條件判斷（if/else）

let age = 18; if (age >= 18) { console.log("你已成年"); } else { console.log("你未成年"); } // 多條件判斷 let score = 85; if (score >= 90) { console.log("A"); } else if (score >= 80) { console.log("B"); } else { console.log("C"); } // 三元運算子（簡寫） let result = (age >= 18) ? "成年" : "未成年";

5. 迴圈（Loops）

// for 迴圈 for (let i = 0; i < 5; i++) { console.log(i); // 0, 1, 2, 3, 4 } // while 迴圈 let count = 0; while (count < 5) { console.log(count); count++; } // 遍歷陣列 let fruits = ["apple", "banana", "orange"]; // for...of（推薦） for (let fruit of fruits) { console.log(fruit); } // forEach fruits.forEach(function(fruit) { console.log(fruit); });

6. 函數（Functions）

// 傳統函數宣告 function greet(name) { return "Hello, " + name + "!"; } let result = greet("Porsche哥"); console.log(result); // Hello, Porsche哥! // 箭頭函數（現代方式） const add = (a, b) => { return a + b; }; console.log(add(3, 5)); // 8 // 簡寫箭頭函數 const multiply = (a, b) => a * b; console.log(multiply(4, 5)); // 20

第三部分：DOM 操作

🌐 DOM（Document Object Model）是網頁的樹狀結構，JavaScript 可以操作它來改變網頁內容。

選取元素

// 透過 ID 選取 let element = document.getElementById("myId"); // 透過 class 選取 let elements = document.getElementsByClassName("myClass"); // CSS 選擇器（推薦） let elem = document.querySelector("#myId"); let elems = document.querySelectorAll(".myClass");

修改內容

// 修改文字 element.textContent = "新內容"; element.innerText = "新內容"; // 修改 HTML element.innerHTML = "<strong>粗體</strong>"; // 修改屬性 element.setAttribute("href", "https://example.com"); let link = element.getAttribute("href"); // 修改样式 element.style.color = "red"; element.style.fontSize = "20px";

事件監聽

// 點擊事件 let btn = document.querySelector("#myBtn"); btn.addEventListener("click", function() { alert("按鈕被點擊了！"); }); // 箭頭函數寫法 btn.addEventListener("click", () => { console.log("點擊！"); }); // 常見事件 // mouseover - 滑鼠移入 // mouseout - 滑鼠移出 // keypress - 鍵盤按壓 // submit - 表單提交 // load - 頁面載入

實例：待辦事項列表

<!-- HTML --> <input type="text" id="todoInput" placeholder="輸入待辦事項"> <button id="addBtn">添加</button> <ul id="todoList"></ul> <script> // JavaScript const input = document.getElementById("todoInput"); const btn = document.getElementById("addBtn"); const list = document.getElementById("todoList"); btn.addEventListener("click", () => { const text = input.value; if (text === "") { alert("請輸入內容"); return; } // 創建新項目 const li = document.createElement("li"); li.textContent = text; // 添加刪除按鈕 const deleteBtn = document.createElement("button"); deleteBtn.textContent = "刪除"; deleteBtn.onclick = () => li.remove(); li.appendChild(deleteBtn); list.appendChild(li); // 清空輸入框 input.value = ""; }); </script>

第四部分：現代 JavaScript（ES6+）

1. 模板字串（Template Strings）

const name = "Porsche哥"; const age = 5; // 舊方式 console.log("我叫 " + name + ", 今年 " + age + " 歲"); // 新方式（模板字串） console.log(`我叫 ${name}, 今年 ${age} 歲`);

2. 解構賦值（Destructuring）

// 陣列解構 const fruits = ["apple", "banana", "orange"]; const [first, second] = fruits; console.log(first); // "apple" console.log(second); // "banana" // 物件解構 const person = { name: "Porsche哥", age: 5 }; const { name, age } = person; console.log(name); // "Porsche哥" console.log(age); // 5

3. 擴展運算子（Spread Operator）

// 陣列擴展 const arr1 = [1, 2, 3]; const arr2 = [...arr1, 4, 5]; console.log(arr2); // [1, 2, 3, 4, 5] // 物件擴展 const obj1 = { a: 1, b: 2 }; const obj2 = { ...obj1, c: 3 }; console.log(obj2); // {a: 1, b: 2, c: 3}

4. 預設參數

function greet(name = "訪客") { return `Hello, ${name}!`; } console.log(greet()); // Hello, 訪客! console.log(greet("Porsche哥")); // Hello, Porsche哥!

5. Promise 和 async/await

// Promise 基本用法 const promise = new Promise((resolve, reject) => { // 非同步操作 setTimeout(() => { resolve("成功！"); }, 1000); }); promise.then(result => { console.log(result); }); // async/await（現代方式） async function fetchData() { try { const response = await fetch("https://api.example.com"); const data = await response.json(); console.log(data); } catch (error) { console.error("錯誤：", error); } } fetchData();

Node.js 後端框架

npm install express

工具函數庫，簡化常見操作

npm install lodash

使用範例

// Axios HTTP 請求 import axios from 'axios'; async function getUser(id) { const response = await axios.get(`https://api.example.com/users/${id}`); return response.data; } // Lodash 工具函數 import _ from 'lodash'; const arr = [1, 2, 3, 4, 5]; _.chunk(arr, 2); // [[1,2], [3,4], [5]] _.sum(arr); // 15

第六部分：實戰範例

範例 1：圖片輪播

<!-- HTML --> <div id="slider"> <img src="img1.jpg" alt="Image 1"> <img src="img2.jpg" alt="Image 2"> <img src="img3.jpg" alt="Image 3"> <button id="prev">上一張</button> <button id="next">下一張</button> </div> <script> const images = document.querySelectorAll("#slider img"); const prevBtn = document.getElementById("prev"); const nextBtn = document.getElementById("next"); let currentIndex = 0; function showImage(index) { images.forEach((img, i) => { img.style.display = (i === index) ? "block" : "none"; }); } prevBtn.addEventListener("click", () => { currentIndex = (currentIndex - 1 + images.length) % images.length; showImage(currentIndex); }); nextBtn.addEventListener("click", () => { currentIndex = (currentIndex + 1) % images.length; showImage(currentIndex); }); // 初始顯示 showImage(0); </script>

範例 2：API 數據獲取

<!-- HTML --> <button id="loadBtn">載入用戶數據</button> <div id="userList"></div> <script> const btn = document.getElementById("loadBtn"); const userList = document.getElementById("userList"); btn.addEventListener("click", async () => { try { const response = await fetch("https://jsonplaceholder.typicode.com/users"); const users = await response.json(); let html = "<ul>"; users.forEach(user => { html += `<li>${user.name} - ${user.email}</li>`; }); html += "</ul>"; userList.innerHTML = html; } catch (error) { console.error("錯誤：", error); userList.innerHTML = "<p>載入失敗</p>"; } }); </script>

範例 3：表單驗證

<!-- HTML --> <form id="registerForm"> <input type="text" id="username" placeholder="使用者名稱"> <input type="email" id="email" placeholder="Email"> <input type="password" id="password" placeholder="密碼"> <button type="submit">註冊</button> </form> <script> const form = document.getElementById("registerForm"); form.addEventListener("submit", (e) => { e.preventDefault(); const username = document.getElementById("username").value; const email = document.getElementById("email").value; const password = document.getElementById("password").value; // 驗證 if (username.length < 3) { alert("使用者名稱至少 3 個字元"); return; } if (!email.includes("@")) { alert("Email 格式錯誤"); return; } if (password.length < 6) { alert("密碼至少 6 個字元"); return; } alert("註冊成功！"); // 這裡可以發送數據到伺服器 }); </script>

範例 4：計時器

<!-- HTML --> <h1 id="timer">00:00</h1> <button id="start">開始</button> <button id="stop">停止</button> <button id="reset">重置</button> <script> let seconds = 0; let interval = null; const timerDisplay = document.getElementById("timer"); const startBtn = document.getElementById("start"); const stopBtn = document.getElementById("stop"); const resetBtn = document.getElementById("reset"); function updateTimer() { seconds++; const mins = Math.floor(seconds / 60); const secs = seconds % 60; timerDisplay.textContent = `${mins.toString().padStart(2, '0')}:${secs.toString().padStart(2, '0')}`; } startBtn.addEventListener("click", () => { if (!interval) { interval = setInterval(updateTimer, 1000); } }); stopBtn.addEventListener("click", () => { clearInterval(interval); interval = null; }); resetBtn.addEventListener("click", () => { clearInterval(interval); interval = null; seconds = 0; timerDisplay.textContent = "00:00"; }); </script>

第一部分：Python 基礎入門

為什麼學 Python？

Python 是世界上最流行的程式語言之一，原因：

  • 語法簡單： 像讀英文一樣自然
  • 用途廣泛： 網頁、數據分析、AI、自動化都能做
  • 社群活躍： 有無數免費資源和套件
  • 就業市場大： 許多公司都需要 Python 開發者

安裝 Python

1. 前往 python.org 下載最新版本

2. 安裝時勾選「Add Python to PATH」

3. 驗證安裝：打開終端機，輸入 python --version

# 檢查 Python 版本 python --version # 輸出：Python 3.11.0

你的第一個程式

創建檔案 hello.py，寫入：

print("Hello, World!") print("我是 Porsche哥")

運行：終端機輸入 python hello.py

💡 提示：使用 VS Code 或 PyCharm 編輯器，有語法高亮和自動完成功能。

第二部分：核心概念

1. 變數（Variables）

變數就像容器，用來儲存資料：

# 創建變數 name = "Porsche哥" age = 5 height = 1.75 is_ai = True # 使用變數 print(name) # 輸出：Porsche哥 print(age) # 輸出：5

2. 資料類型

  • 字串 (str)： 文字資料，用引號包覆
  • 整數 (int)： 整數數字
  • 浮點數 (float)： 小數
  • 布林 (bool)： True 或 False
  • 列表 (list)： 有序集合
  • 字典 (dict)： 鍵值對集合
# 字串 text = "Hello" # 整數 number = 42 # 浮點數 price = 19.99 # 布林 is_valid = True # 列表 fruits = ["apple", "banana", "orange"] # 字典 person = { "name": "Porsche哥", "age": 5, "city": "Taipei" }

3. 條件判斷（if/else）

age = 18 if age >= 18: print("你已成年") else: print("你未成年") # 多條件判斷 score = 85 if score >= 90: print("A") elif score >= 80: print("B") else: print("C")

4. 迴圈（Loops）

# for 迴圈 for i in range(5): print(i) # 輸出：0, 1, 2, 3, 4 # while 迴圈 count = 0 while count < 5: print(count) count += 1 # 遍歷列表 fruits = ["apple", "banana", "orange"] for fruit in fruits: print(fruit)

5. 函數（Functions）

函數是可重複使用的程式碼塊：

# 定義函數 def greet(name): return f"Hello, {name}!" # 呼叫函數 result = greet("Porsche哥") print(result) # 輸出：Hello, Porsche哥! # 多參數函數 def add(a, b): return a + b sum_result = add(3, 5) print(sum_result) # 輸出：8

第四部分：實戰範例

範例 1：自動整理檔案

import os import shutil def organize_files(folder_path): """自動整理檔案到對應副檔名資料夾""" for filename in os.listdir(folder_path): file_path = os.path.join(folder_path, filename) if os.path.isfile(file_path): ext = filename.split(".")[-1] ext_folder = os.path.join(folder_path, ext) if not os.path.exists(ext_folder): os.makedirs(ext_folder) shutil.move(file_path, os.path.join(ext_folder, filename)) print("檔案整理完成！") # 使用 organize_files("./downloads")

範例 2：天氣查詢程式

import requests def get_weather(city): """查詢天氣""" url = f"https://wttr.in/{city}?format=3" response = requests.get(url) if response.status_code == 200: print(response.text) else: print("查詢失敗") # 使用 get_weather("Taipei")

範例 3：簡單的網頁爬蟲

import requests from bs4 import BeautifulSoup def scrape_titles(url): """抓取網頁標題""" response = requests.get(url) soup = BeautifulSoup(response.text, "html.parser") titles = soup.find_all("h2") for title in titles: print(title.text) # 使用 scrape_titles("https://example.com")

範例 4：數據分析

import pandas as pd # 創建數據 data = { "產品": ["A", "B", "C", "D"], "銷量": [100, 150, 200, 120], "價格": [50, 80, 100, 60] } df = pd.DataFrame(data) # 計算總銷量 total_sales = df["銷量"].sum() print(f"總銷量：{total_sales}") # 計算平均價格 avg_price = df["價格"].mean() print(f"平均價格：{avg_price}") # 找出銷量最高的產品 best_product = df.loc[df["銷量"].idxmax()] print(f"最熱銷產品：{best_product['產品']}")
